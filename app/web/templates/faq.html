Preguntas frecuentes

Guía rápida para operar AutoTest Pro Framework durante las pruebas de estabilización.

¿Cuál es la versión base estable?

La base actual es la Fase 18. Los cambios se trabajan como correcciones incrementales sobre esa versión.

¿Cómo inicio la aplicación localmente?

Activa el entorno virtual, instala dependencias, inicializa SQLite y ejecuta python api_server.py.

¿Dónde ingreso a la plataforma?

Abre http://127.0.0.1:8000/login e ingresa con el usuario administrador inicial.

¿Dónde veo la documentación técnica de la API?

Entra a /docs para usar Swagger y probar endpoints de forma controlada.

¿Cómo ejecuto una prueba grabada?

Ingresa al Grabador, selecciona un caso, valida sus pasos y presiona ejecutar. Luego revisa el visor avanzado de ejecución.

¿Cómo se capturan acciones automáticamente?

Instala la extensión local de Chrome, configura el Case ID y API base, activa la captura y navega en el sistema bajo prueba.

¿Cómo limpio pasos duplicados del grabador?

Desde el módulo Grabador usa la opción de limpieza de duplicados antes de ejecutar o exportar el caso.

¿Cómo exporto un caso a Page Object?

Desde el Grabador abre el caso, usa Exportar Page Object, revisa la vista previa y descarga el ZIP generado.

¿Dónde se generan los reportes?

Los reportes se guardan en las carpetas configuradas para HTML, Excel y PDF. También pueden abrirse desde el módulo Reportes.

¿Qué hago si no aparecen reportes?

Ejecuta primero un caso demo o un caso grabado. Luego entra al módulo Reportes y actualiza la página.

¿Cómo cambio el idioma?

Usa el selector de idioma en la barra superior o el campo Idioma en Configuración. El cambio se aplica a los textos principales de la interfaz.

¿Cómo cambio el tema visual?

Usa el selector de tema en la barra superior o Configuración. El tema queda guardado y se aplica al recargar.

¿Puedo usar la librería sin la interfaz?

Sí. El framework está separado de la interfaz y puede ejecutarse desde consola o integrarse mediante API.

¿Qué información debo enviar al reportar un bug?

Comparte captura, pasos para reproducir, consola del servidor, navegador usado y módulo donde ocurrió.

¿Cuándo seguimos con nuevas fases?

Cuando esta base quede estable con los bugs y mejoras corregidos, se continúa con la siguiente fase funcional.
