- Se reemplazó el dashboard básico por una vista ejecutiva más completa.
- Se agregaron nuevas tarjetas de métricas: proyectos activos, casos grabados, ejecuciones, reportes, jobs, fallidas y duración promedio.
- Se agregaron gráficos adicionales: distribución de resultados, ejecuciones por navegador y tendencia reciente.
- Se quitó la ejecución demo del dashboard.
- Se agregó botón hacia el último reporte generado.
- Se conserva acceso a ejecuciones desde el dashboard.

- Se agregó acción `Abrir / Ver`.
